# Nova Core — Releases

Binary releases of **Nova Core**, a dependency-free DVB to MPEG-TS streaming headend.

- Single static binary, Linux x86-64. No ffmpeg, no libraries.
- Install as a systemd service: `sudo ./nova-core -install`
- Verify every download against its published `nova-core.sha256`.

Source is private; this repository hosts compiled binaries only.
